# 🦀 LeetCode × Rust

A personal repository for solving LeetCode problems in Rust — built to learn the language deeply while sharpening problem-solving skills.

## Project Structure

```
leetcode-rust/
├── src/
│   ├── lib.rs                   # Crate root
│   ├── list_node.rs             # ListNode definition (for linked list problems)
│   └── problems/
│       ├── mod.rs               # Module declarations
│       └── p{number}_{name}.rs  # One file per problem
└── Cargo.toml
```

Each file contains the solution along with test cases.
